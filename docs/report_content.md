## 作品题目

**随机数的统计分布测试**

## 作品内容摘要

本作品针对C语言中常用的 `rand()%N` 随机数生成方法进行系统性统计分布测试。实现了6种测评工具，分别从频率分布、重复间隔分布、卡方拟合优度检验、游程检验、Kolmogorov-Smirnov检验和自相关检验等多个维度评估该方法的随机性质。同时，设计了均匀分布和正态分布两种随机数生成器，并使用中科国昱密评工具箱对生成的随机数序列进行测评。实验结果通过可视化图表直观展示，为密码学中的随机数质量评估提供了完整的技术方案。

## 2. 设计与实现方案

### 2.1 实现原理

#### 2.1.1 rand() % N 的原理与缺陷

C语言标准库的 `rand()` 函数基于线性同余生成器(LCG)：

```
seed = (seed × 1103515245 + 12345) mod 2³¹
```

LCG生成器的低位周期较短，当N为2的幂次时，`rand()%N` 的低位分布不均匀。此外，当 RAND_MAX+1 不能被 N 整除时，模运算会产生偏差（modulo bias）。

#### 2.1.2 均匀分布生成原理

**Box-Muller变换法**：
设 U₁, U₂ 是 (0,1] 上的独立均匀分布随机数，则：
- Z₀ = √(-2·ln U₁) · cos(2πU₂)
- Z₁ = √(-2·ln U₁) · sin(2πU₂)
Z₀, Z₁ 服从标准正态分布 N(0,1)。

**中心极限定理法**：
根据中心极限定理，n个独立同分布的随机变量之和近似服从正态分布。
对于 [0,1) 均匀分布 Uᵢ，E[Uᵢ]=1/2, Var[Uᵢ]=1/12。
Z = (ΣUᵢ - n/2) / √(n/12) 近似服从 N(0,1)。

#### 2.1.3 各测评工具的数学原理

**① 频率分布统计**：
统计每个数值 v ∈ [0,N-1] 的出现频次 f(v) 和概率 p(v)=f(v)/n。
理论均匀概率为 1/N，计算最大偏差和平均绝对偏差。

**② 重复间隔分布统计**：
对每个数值 v，记录其相邻出现位置的距离 dᵢ = posᵢ₊₁ - posᵢ。
统计所有间隔的均值、中位数、标准差。

**③ 卡方拟合优度检验**：
χ² = Σᵢ (Oᵢ - Eᵢ)² / Eᵢ
其中 Oᵢ 为观测频数，Eᵢ = n/N 为期望频数。
χ² 服从自由度为 N-1 的卡方分布。

**④ 游程检验（Wald-Wolfowitz）**：
将序列按中位数分为两类，统计游程总数 R。
期望游程数 E[R] = 2n₀n₁/(n₀+n₁) + 1
Z = (R - E[R]) / σᴿ ∼ N(0,1)

**⑤ KS检验**：
D = max |Fₙ(x) - F(x)|
其中 Fₙ(x) 为经验分布函数，F(x) 为理论均匀分布函数。

**⑥ 自相关检验（Ljung-Box Q）**：
Q = n(n+2) Σₖ rₖ²/(n-k)
其中 rₖ 为滞后 k 的自相关系数，Q ∼ χ²(max_lag)。

## 3. 系统测试与结果

### 3.4 测试结论

在 N=100, 样本量=10000 的测试条件下，`rand()%N` 方法通过了4/6种测评工具的检验：

| 测评工具 | p-value | α=0.05 | 结论 |
|---------|---------|--------|------|
| 卡方检验 | 0.074 | p > α | ✅ 接受均匀假设 |
| 游程检验 | 0.246 | p > α | ✅ 通过随机性检验 |
| KS检验 | **0.048** | p < α | ❌ **拒绝均匀假设** |
| 自相关检验 | 0.612 | p > α | ✅ 通过独立性检验 |

**分析与讨论**：

1. **KS检验未通过的原因**：KS检验的 p=0.0483 略低于显著性水平 α=0.05，属于"边缘显著"情况。这并非意味着 `rand()%N` 存在严重缺陷，而是反映了两方面因素：(a) KS检验对经验分布与理论分布的最大偏差敏感，当样本量增大到10000时，检验力(power)增强，微小的偏差也能被检测出来；(b) LCG生成器固有周期为2³¹，低位周期较短，模运算引入的系统性偏差在高样本量下被KS检验捕捉到。

2. **卡方检验与KS检验的差异**：卡方检验(p=0.074)通过了而KS检验(p=0.048)未通过，这是因为两种检验的侧重点不同——卡方检验衡量离散频数的整体拟合度，而KS检验聚焦于累积分布的最大偏差。LCG的低位周期性在累积分布上产生了略微更大的局部偏差，被KS检验检测到，但该偏差尚不足以影响卡方检验的结果。

3. **自相关检验的发现**：在50个滞后阶数中，有3个滞后（14, 35, 43）的自相关系数超过了95%置信阈值。这是LCG线性递归结构的固有特征——相邻随机数之间存在一定的相关性。但整体Q统计量不显著(p=0.612)，表明自相关程度很弱。

4. **均匀生成器对比结果**：四种均匀分布生成方法的性能排序为：**自定义LCG > 拒绝采样法 > 浮点乘法律 > rand()%N**。自定义LCG的p值最高(0.920)，偏差最小；而 `rand()%N` 表现最差，验证了模偏差(modulo bias)理论分析的正确性。

5. **正态分布生成器验证**：Box-Muller变换法和中心极限定理法均通过了Shapiro-Wilk正态性检验(p>0.05)，说明两种方法都能有效生成服从正态分布的随机数。

**综合评价**：`rand()%N` 在频率分布、序列随机性和独立性方面表现良好，但在累积分布均匀性方面存在边缘性缺陷。对于非密码学应用（如游戏、简单模拟），`rand()%N` 仍然可用；但对于密码学、金融模拟等对随机性要求严格的应用场景，建议使用更高质量的随机数生成器（如拒绝采样法或专用密码学安全的伪随机数生成器）。

## 5. 结论

本作品实现了对 `rand()%N` 随机数生成方法的系统性统计分布测试，包含6种专业测评工具和2种分布生成器。通过理论分析和实验验证，全面评估了该方法的随机质量。

实验结果表明（N=100, 样本量=10000）：
- **卡方检验**：χ²=120.08, p=0.074，通过均匀分布假设检验
- **游程检验**：Z=-1.16, p=0.246，通过序列随机性检验
- **KS检验**：D=0.0136, **p=0.048，未通过均匀分布假设检验**（边缘显著）
- **自相关检验**：Q=46.56, p=0.612，通过独立性检验

总体而言，`rand()%N` 在频率分布、序列随机性和独立性方面表现良好，但KS检验揭示了其在累积分布均匀性方面存在边缘性缺陷，验证了LCG算法低位周期短和模偏差(modulo bias)等固有限制。四种均匀分布生成方法的性能排序为：自定义LCG > 拒绝采样法 > 浮点乘法律 > rand()%N。在密码学等安全性要求高的场景中，建议使用拒绝采样法或密码学安全的伪随机数生成器替代 `rand()%N`。
