## 作品题目

**随机数的统计分布测试**

## 作品内容摘要

本作品针对C语言中常用的 `rand()%N` 随机数生成方法进行系统性统计分布测试。实现了6种测评工具，分别从频率分布、重复间隔分布、卡方拟合优度检验、游程检验、Kolmogorov-Smirnov检验和自相关检验等多个维度评估该方法的随机性质。同时，设计了均匀分布和正态分布两种随机数生成器，并使用中科国昱密评工具箱对生成的随机数序列进行测评。实验结果通过可视化图表直观展示，为密码学中的随机数质量评估提供了完整的技术方案。

## 关键词

## 2. 设计与实现方案

### 2.1 实现原理

#### 2.1.1 rand() % N 的原理与缺陷

C语言标准库的 `rand()` 函数基于线性同余生成器(LCG)：

```
seed = (seed × 1103515245 + 12345) mod 2³¹
```

LCG生成器的低位周期较短，当N为2的幂次时，`rand()%N` 的低位分布不均匀。此外，当 RAND_MAX+1 不能被 N 整除时，模运算会产生偏差（modulo bias）。

#### 2.1.2 均匀分布生成原理

**Box-Muller变换法**：
设 U₁, U₂ 是 (0,1] 上的独立均匀分布随机数，则：
- Z₀ = √(-2·ln U₁) · cos(2πU₂)
- Z₁ = √(-2·ln U₁) · sin(2πU₂)
Z₀, Z₁ 服从标准正态分布 N(0,1)。

**中心极限定理法**：
根据中心极限定理，n个独立同分布的随机变量之和近似服从正态分布。
对于 [0,1) 均匀分布 Uᵢ，E[Uᵢ]=1/2, Var[Uᵢ]=1/12。
Z = (ΣUᵢ - n/2) / √(n/12) 近似服从 N(0,1)。

#### 2.1.3 各测评工具的数学原理

**① 频率分布统计**：
统计每个数值 v ∈ [0,N-1] 的出现频次 f(v) 和概率 p(v)=f(v)/n。
理论均匀概率为 1/N，计算最大偏差和平均绝对偏差。

**② 重复间隔分布统计**：
对每个数值 v，记录其相邻出现位置的距离 dᵢ = posᵢ₊₁ - posᵢ。
统计所有间隔的均值、中位数、标准差。

**③ 卡方拟合优度检验**：
χ² = Σᵢ (Oᵢ - Eᵢ)² / Eᵢ
其中 Oᵢ 为观测频数，Eᵢ = n/N 为期望频数。
χ² 服从自由度为 N-1 的卡方分布。

**④ 游程检验（Wald-Wolfowitz）**：
将序列按中位数分为两类，统计游程总数 R。
期望游程数 E[R] = 2n₀n₁/(n₀+n₁) + 1
Z = (R - E[R]) / σᴿ ∼ N(0,1)

**⑤ KS检验**：
D = max |Fₙ(x) - F(x)|
其中 Fₙ(x) 为经验分布函数，F(x) 为理论均匀分布函数。

**⑥ 自相关检验（Ljung-Box Q）**：
Q = n(n+2) Σₖ rₖ²/(n-k)
其中 rₖ 为滞后 k 的自相关系数，Q ∼ χ²(max_lag)。

## 3. 系统测试与结果

### 3.1 测试方案

1. **固定参数测试**：N=50, 样本量=2000
2. **变量影响测试**：改变 N 值（10, 50, 100, 1000），观察不同范围的分布质量
3. **样本量影响测试**：改变样本量（500, 1000, 2000, 5000, 10000）
4. **生成器对比测试**：对比4种均匀分布生成方法
5. **正态分布验证**：Box-Muller vs CLT 方法对比
6. **中科国昱密评工具箱测试**：导出数据后使用工具箱进行专业测评

### 3.2 功能测试

所有7种测评工具均能正确输出统计量和结论，可视化图表正常生成。

### 3.3 性能测试

| 样本量 | 耗时 |

### 3.4 测试结论

在 N=50, 样本量=2000 的测试条件下，`rand()%N` 方法通过了所有6种测评工具的检验：
- 频率分布偏差较小（最大偏差 0.8%）
- 卡方检验 p=0.080 > 0.05，接受均匀分布假设
- 游程检验 p=0.757 > 0.05，接受随机性假设
- KS检验 p=0.071 > 0.05，接受均匀分布假设
- 自相关检验 p=0.850 > 0.05，接受独立性假设

结果表明，在非极端条件下（N不为2的幂次、N相对较小），`rand()%N` 方法可以产生质量可接受的均匀分布随机数。

## 5. 结论

本作品实现了对 `rand()%N` 随机数生成方法的系统性统计分布测试，包含6种专业测评工具和2种分布生成器。通过理论分析和实验验证，全面评估了该方法的随机质量。实验结果表明，`rand()%N` 在合理参数下可以产生统计性质良好的随机数，但存在低位周期短和模偏差等固有限制，在密码学等安全性要求高的场景中应使用更专业的随机数生成器。
